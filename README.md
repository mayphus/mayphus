## 💻 About This Project (mayphus.org)

This repository contains the source code for my personal website, built with Angular and deployed on Firebase.

### Development

To run this project locally:

1.  **Install dependencies:**
    ```bash
    pnpm install
    ```

2.  **Start the development server:**
    ```bash
    pnpm start
    ```
    Navigate to `http://localhost:4200/`.

3.  **Build:**
    ```bash
    pnpm build
    ```

### Deployment

This project is automatically deployed to Firebase Hosting via GitHub Actions on pushes to the `main` branch.
